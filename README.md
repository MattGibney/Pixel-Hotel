## No AI Development

I work with AI a lot these days. I adopted it fairly early but with the agentic
coding being as good as it is, it's a huge part now of my daily life. In my
day-to-day work, I mostly review code and direct agents rather than write it
myself.

AI has made software development much cheaper, which, I for the most part really
appreciate. But one of the big reasons that I was so interested in code wasn't
to ship features, It was because I really enjoy coding. This is a project for me
to have something interesting and challenging to work on that can help me to
enjoy programming once again.
